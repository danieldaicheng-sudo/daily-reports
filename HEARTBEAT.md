- **执行时间**: 每天 21:00（北京时间）
- **状态**: idle（正常）

## 任务调度规则

### 任务1：每日工作日报（21:00）
**触发条件**：每天 21:00
**执行内容**：
1. **读取所有 Chrome 浏览器中的 Claude 标签页**（通过 browser tabs + snapshot）
   - 用户每天在 3 个不同的 Claude 标签页中工作
   - 必须获取所有 3 个标签页的内容
   - 整合分析所有标签页的工作内容
2. **分析当天工作内容**，识别：
   - 今日完成的关键产出（不是罗列，而是提炼认知增量）
   - 重要数据/发现/决策
   - 待办事项状态变化
3. **生成结构化日报**，包含：
   - 【今日核心认知】提炼今天工作带来的关键洞察
   - 【关键产出】价值导向的描述，说明"做了什么"和"为什么重要"
   - 【决策/调整】流程优化、策略调整等
   - 【明日待办】下一步行动
   - 【团队协作点】需要同步给团队的信息
4. **覆盖规则**：如 memory/YYYY-MM-DD.md 已存在，直接覆盖（重新生成最新版本）

### 任务2（已合并）：推送日报到GitHub
**状态**：已合并到任务1
**执行内容**：日报生成后自动执行 git add → commit → push 到 origin master

### 任务3：周六工作周报（18:00）
**触发条件**：每周六 18:00
**执行内容**：
1. 汇总本周 6 份日报
2. 提炼周度核心认知：
   - 本周工作的战略价值（对港险AI Agent项目的贡献）
   - 方法论沉淀（优化流程、工作规范迭代）
   - 数据/洞察积累（可用于团队知识库）
3. 生成团队对齐周报，包含：
   - 【本周核心成果】对项目的实际推进
   - 【关键认知/洞察】可以复用的方法论
   - 【下周计划】重点方向和预期产出
   - 【需团队协同事项】需要其他成员配合/知晓的内容

## 执行规范

### 1. 内容质量标准（日报必须满足）

**🚫 禁止事项（这些内容写出来是浪费团队时间）：**
- ❌ 单纯罗列"我做了A、做了B、做了C"
- ❌ 流水账式的时间线记录
- ❌ 没有上下文的技术细节堆砌

**✅ 必须包含（团队真正想看的）：**
- **🧠 认知沉淀**：今天我理解了什么？发现了什么规律？推翻了什么之前的假设？
- **🎯 问题解决**：遇到了什么坑？怎么解决的？别人能怎么避免？
- **📊 方法论输出**：有什么可以复用的流程、工具、checklist？
- **🤝 协作需求**：我需要谁的帮助？团队需要知道什么？有什么阻塞？
- **🚀 明日聚焦**：明天最重要的1-3件事是什么？预期产出是什么？

### 2. 团队价值导向
- 写完后问自己：**"如果我是团队其他成员，读了这个日报能获得什么？"**
- 如果答案只是"知道他今天很忙"，那就重写
- 好的日报让其他人能：**学习经验**、**避免踩坑**、**主动协作**

2. **Chrome 访问**：
   - 使用 browser tabs 获取所有 Claude 页面
   - 使用 snapshot 读取每个页面的关键内容
   - 重点关注：
     * 完成的任务/达成的里程碑
     * 重要发现/数据/决策
     * 待办事项的变化

3. **输出格式**：
   - 日报：简洁的 Markdown，适合个人回顾
   - 周报：正式的 Markdown，适合团队共享

## 示例模板

### 日报示例结构
```
【2026-03-17 工作日报】

🧠 今日核心认知
今天最重要的1-3个洞察/发现/理解变化是什么？
- 不是做了什么，而是"我懂了什么"
- 可以复用到其他场景的认知

🎯 问题解决与踩坑
今天遇到了什么难题？怎么解决的？
- 问题：[描述问题现象/背景]
- 根因：[为什么会遇到这个问题]
- 解决：[采取了什么措施]
- 教训：[别人如何避免/如何更快解决]

💡 方法论沉淀
今天有什么可以复用的产出？
- 流程优化：改进了什么工作方式？
- 工具/脚本：写了什么可以复用的代码？
- Checklist/规范：总结了什么可复制的标准？

📊 关键产出（选填）
如果今天有具体的交付物，简要说明：
- 产出名称 + 对项目的价值（为什么重要）
- 关键数据/指标（如果有）

🚀 明日聚焦
明天最重要的1-3件事：
- 事项 + 预期产出（什么是"完成"的标准）

🤝 团队协同
- 我需要团队什么支持？
- 团队需要知道什么信息？
- 有什么阻塞需要同步？
```

### 周报示例结构

```
【2026年第X周 工作周报】

[对港险AI Agent知识库项目的实际推进]

🎯 下周计划
[重点方向和预期产出]

🤝 需团队协同
[需要其他成员配合/知晓的事项]
```
